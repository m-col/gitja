    1  gitserve
    2  ========
    3  
    4  .. image:: https://img.shields.io/github/v/release/m-col/gitserve?color=904ff0
    5     :alt: GitHub release (latest SemVer)
    6  
    7  
    8  ``gitserve`` generates pretty web pages from your git repositories using custom
    9  Jinja_\* HTML templates.
   10  
   11  ``gitserve`` complements your static site generator so that you can display
   12  your code however you want, in a style consistent with the rest of your
   13  website!
   14  
   15  This was inspired by stagit_, which generates content with a hard-coded style,
   16  so if you like that style and don't want to edit templates, you may prefer to
   17  use that.
   18  
   19  \*The templating engine is not Jinja per se, but ginger_, an implementation of
   20  the Jinja language for Haskell. Please see their docs for any differences.
   21  
   22  Getting Started
   23  ---------------
   24  
   25  Currently installation is only possible via source::
   26  
   27      git clone https://github.com/m-col/gitserve
   28      cd gitserve
   29      stack install
   30  
   31  Documentation and a preview of the kind of pages ``gitserve`` can generate is
   32  available here_.
   33  
   34  If you like skipping the manual, running ``gitserve --template`` will create a
   35  base template and config in the current folder from which you can create your
   36  own.
