    1  gitserve
    2  ========
    3  
    4  .. image:: https://img.shields.io/github/v/release/m-col/gitserve?color=904ff0
    5     :alt: GitHub release (latest SemVer)
    6  
    7  
    8  ``gitserve`` generates pretty web pages from your git repositories using custom
    9  Jinja_\* HTML templates.
   10  
   11  This was inspired by stagit_, which generates content with a hard-coded style,
   12  so if you like that style and don't want to edit templates, you may prefer to
   13  use that.
   14  
   15  \*The templating engine is not Jinja per se, but ginger_, an implementation of
   16  the Jinja language for Haskell. Please see their docs for any differences.
   17  
   18  Why?
   19  ----
   20  
   21  ``gitserve`` complements your static site generator so that you can display
   22  your code however you want, in a style consistent with the rest of your
   23  website!
   24  
   25  Contributing
   26  ------------
   27  
   28  If you've come across a bug or have a question feel free to `open an issue`_.
   29  Pull requests are also warmly welcome if you have any suggested improvements or
   30  fixes. Please format code with fourmolu_'s defaults, use hlint_, and make sure
   31  ``test/test.sh`` is happy.
